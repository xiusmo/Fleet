分布式任务调度平台 + 用户 & 管理员前端界面 | Docker Compose 部署

项目概览

Fleet 是一个基于分布式架构的任务调度平台，提供任务提交、调度、状态监控及管理后台，包含用户前端与管理员前端，支持一键 Docker Compose 部署。

核心特性

  • Python FastAPI API 服务（RESTful，JWT 认证）
  • 分布式调度器：异步任务执行、工作节点心跳
  • Vue 3 + Vite 用户前端
  • Vue 3 + Element Plus 管理员后台
  • Nginx 反向代理 & ModSecurity 防火墙
  • PostgreSQL 数据持久化 + pgAdmin 管理
  • 自动 SSL/TLS 证书管理（Certbot & Let's Encrypt）
  • 容器自动更新（Watchtower）
  • 日志持久化与健康检查

技术栈

组件技术
后端Python 3.11, FastAPI, SQLModel, asyncpg
调度器Python, asyncio
用户前端Vue 3, Vite, Axios
管理后台Vue 3, Element Plus, Pinia
数据库PostgreSQL
容器编排Docker Compose
反代 & 安全Nginx, ModSecurity, Certbot

项目结构

Fleet/
├─ fleet_task_scheduler/   # API & 调度服务 (FastAPI) 被软件多实例部署
├─ fleet_node_service/     # 子节点 (FastAPI) 被物理多实例部署
├─ user_frontend/          # 用户前端 (Vue 3 + Vite)
├─ admin_dashboard/        # 管理后台 (Vue 3 + Element Plus)
├─ nginx/                  # Nginx 配置 & 前端静态文件
├─ docker-compose.yml      # 服务编排文件
├─ .env                    # 环境变量示例
└─ README.md               # 项目说明文档
